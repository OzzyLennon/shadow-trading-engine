## 🧠 AI 大脑 v2.0 深度优化

### 数据源升维：多模态输入

| 数据源 | 用途 | 影响 |
|--------|------|------|
| **美股三大指数** | 隔夜外围情绪 | A股开盘重要参考 |
| **富时A50期指** | A股风向标 | 直接影响开盘 |
| **两市成交额** | 市场活跃度 | <7000亿防守, >1.5万亿进攻 |

| **早盘新闻** | 政策热点 | 板块轮动依据 |

### 思维链推理 (Chain of Thought)

```xml
<thinking>
第一步 (宏观定调)：分析外盘+资金面，判断风险等级
第二步 (主线挖掘)：从新闻提取热点板块
第三步 (参数映射)：推导 surge_threshold, stop_loss_pct, trade_ratio
第四步 (反思调整)：根据昨日盈亏调整策略
第五步 (动态选股)：选择 6-10 只标的
</thinking>

<output>
{ "market_sentiment": "...", "symbols": {...} }
</output>
```

### 反馈闭环

```
昨日盈亏 → AI 反思 → 今日参数调整
```

- 昨日亏损 → 降低 trade_ratio，提高 surge_threshold
- 昨日盈利 → 可适当激进，但不超过风险承受

## ⚡ v4.1 排雷优化

根据投资顾问高频守护进程实战指南，v4.1 新增三大防线：

### 防线 1：EMA 滤波器（过滤微观噪音）

```
EMA_t = α × price_t + (1-α) × EMA_{t-1}
```

- α = 0.3（平滑系数）
- 可阻挡 80% 极其恶劣的"骗炮"信号

### 防线 2：交易冷却期状态锁

| 冷却类型 | 时长 | 说明 |
|----------|------|------|
| 股票级 | 5 分钟 | 买入后该股票进入冷却 |
| 账户级 | 2 分钟 | 买入后账户禁止新买入 |

### 防线 3：严格交易时段 + 脏数据拦截

| 检查项 | 拦截条件 |
|--------|----------|
| 价格下限 | price < 1元 |
| 除零保护 | price <= 0 |
| 异常波动 | 单次变化 > 20% |
| 交易时段 | 09:30-11:30, 13:00-14:57 |

## 🧪 统计学核心（LaTeX 精确公式）

### 1. Z-Score 波动率调整

```
Z = (r_t - μ) / σ
```

当 `Z > 2.0` 时，突破 95% 置信区间，触发买入。

### 2. 凯利公式仓位管理

```
f* = (bp - (1-p)) / b
```

- `b` = 赔率（盈亏比）
- `p` = 胜率
- 使用半凯利策略，更保守

### 3. VaR 风险预警

```
VaR_α = inf { L : P(Loss > L) ≤ 1-α }
```

99% 置信度下的最大可能损失。

## 🚀 快速开始

### 1. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 填入你的 API Key
```

### 2. 运行 AI 大脑（每天 8:30）

```bash
python ai_brain.py
```

### 3. 启动守护进程

```bash
# 前台运行
python apex_quant_simulator.py

# 后台运行
nohup python apex_quant_simulator.py > logs/daemon.log 2>&1 &
```

### 4. 停止守护进程

```bash
kill <PID>  # 发送 SIGTERM 优雅退出
```

## ⚙️ 参数配置

在 `apex_quant_simulator.py` 头部修改：

```python
# 轮询间隔
POLL_INTERVAL = 5  # 秒，建议 3-5

# EMA 滤波
EMA_ALPHA = 0.3  # 越小越平滑

# 冷却期
COOLDOWN_MINUTES = 5  # 股票级
ACCOUNT_COOLDOWN_MINUTES = 2  # 账户级

# 交易时段
MORNING_START = (9, 30)
MORNING_END = (11, 30)
AFTERNOON_START = (13, 0)
AFTERNOON_END = (14, 57)  # 避开尾盘集合竞价
```
