## 🧪 统计学核心升级

### 1. 信号生成层：从"固定阈值"到"统计分布"

#### 1.1 Z-Score 波动率调整信号
传统方法：`velocity = (price - prev) / prev`

**升级后**：
- 计算滚动窗口收益率序列
- 计算均值 μ 和标准差 σ
- **买入信号**：当 `(r - μ) / σ > 2.0` 时触发
  - 表示突破 95% 置信区间，属于真正的"极端资金点火"
  - 市场波动率高时自动提高阈值，过滤假突破

#### 1.2 布林带 (Bollinger Bands) 动量
- 中轨 = 20日移动平均
- 上轨/下轨 = 中轨 ± 2倍标准差
- **买入信号**：价格突破布林带上轨

### 2. 仓位管理层：凯利公式

根据历史交易统计自动计算最优仓位：

```
f* = (bp - q) / b

其中：
b = 赔率（盈亏比）
p = 胜率
q = 1 - p
```

**使用半凯利**：系统使用 `f* × 0.5`，更加保守稳健。

每只股票的仓位独立计算，随着交易次数增加自动优化。

### 3. 风险控制层：VaR 预警

**VaR (Value at Risk)**：在给定置信区间（99%）下，未来一个交易日的最大可能损失。

当 VaR 超过组合价值的 3% 时，系统会发出预警。

## 🚀 快速开始

### 1. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 填入你的 API Key
```

### 2. 安装依赖

```bash
pip install requests
```

### 3. 运行

```bash
# 先运行大脑（生成当日参数）
python ai_brain.py

# 再运行统计引擎（执行交易）
python apex_quant_simulator.py
```

### 4. 配置定时任务

```bash
# 每天 8:30 运行大脑
30 8 * * 1-5 cd /path/to/shadow-trading-engine && /usr/bin/python3 ai_brain.py >> logs/brain.log 2>&1

# 交易时段每 2 分钟运行引擎
*/2 9-15 * * 1-5 cd /path/to/shadow-trading-engine && /usr/bin/python3 apex_quant_simulator.py >> logs/apex.log 2>&1
```

## 📊 策略说明

### 买入策略（统计信号）
| 信号类型 | 触发条件 | 说明 |
|----------|----------|------|
| Z-Score | > 2.0 | 突破 95% 置信区间 |
| 布林带 | 突破上轨 | 动量加速 |
| 涨速 | > 1.5% | 备用固定阈值 |

### 卖出策略
| 策略 | 触发条件 | 说明 |
|------|----------|------|
| 追踪止盈 | 盈利≥5%后回落3% | 锁定利润 |
| 铁血止损 | 亏损达到 8% | 硬止损 |

### 仓位管理
| 方法 | 说明 |
|------|------|
| AI 建议比例 | 根据新闻情绪动态调整 |
| 凯利公式 | 根据历史胜率/赔率优化 |
| 最终仓位 | 取两者最小值（保守策略） |

### 交易成本
| 成本项 | 费率 | 方向 |
|--------|------|------|
| 滑点 | 0.2% | 双向 |
| 印花税 | 0.1% | 仅卖出 |
| 佣金 | 0.025% | 双向 |
