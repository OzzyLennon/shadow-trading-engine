## 🧪 统计学核心升级（LaTeX 精确公式）

### 1. 信号生成层：Z-Score 波动率调整

**LaTeX 公式**：
```
Z = (r_t - μ) / σ
```

其中：
- `r_t` = 最新收益率
- `μ` = 收益率序列均值
- `σ` = 收益率序列标准差

**触发条件**：当 `Z > 2.0` 时，突破 95% 置信区间。

**优势**：市场波动率高时自动提高阈值，过滤假突破。

### 2. 仓位管理层：凯利公式

**LaTeX 公式**：
```
f* = (bp - (1-p)) / b
```

其中：
- `b` = 赔率（盈亏比）= 平均盈利 / 平均亏损
- `p` = 胜率
- `1-p` = 败率

**实现细节**：
- 每只股票独立统计胜率和赔率
- 使用半凯利策略（`f* × 0.5`），更保守
- 当 `f* ≤ 0` 时，策略期望为负，不开火

### 3. 风险控制层：VaR (Value at Risk)

**LaTeX 公式**：
```
VaR_α = inf { L ∈ ℝ : P(Loss > L) ≤ 1 - α }
```

在 99% 置信区间下，未来一个交易日的最大可能损失。

**实现方法**：历史模拟法
- 对历史收益率排序
- 取 1% 分位数对应的损失值
- VaR = 组合价值 × 最大损失率

## 🚀 快速开始

### 1. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 填入你的 API Key
```

### 2. 安装依赖

```bash
pip install requests
```

### 3. 运行

```bash
# 先运行大脑（生成当日参数）
python ai_brain.py

# 再运行统计引擎（执行交易）
python apex_quant_simulator.py
```

### 4. 配置定时任务

```bash
# 每天 8:30 运行大脑
30 8 * * 1-5 cd /path/to/shadow-trading-engine && /usr/bin/python3 ai_brain.py >> logs/brain.log 2>&1

# 交易时段每 2 分钟运行引擎
*/2 9-15 * * 1-5 cd /path/to/shadow-trading-engine && /usr/bin/python3 apex_quant_simulator.py >> logs/apex.log 2>&1
```

## 📊 策略说明

### 买入策略（三重信号验证）
| 信号类型 | 公式 | 触发条件 |
|----------|------|----------|
| Z-Score | `(r_t - μ) / σ` | > 2.0 |
| 布林带 | 价格 vs 上轨 | 突破上轨 |
| 涨速 | `(P_t - P_0) / P_0` | > 1.5% |

### 卖出策略
| 策略 | 条件 |
|------|------|
| 追踪止盈 | 盈利≥5% 后回落 3% |
| 铁血止损 | 亏损达到 8% |

### 仓位管理
```
实际仓位 = min(AI建议比例, 凯利建议比例) × 0.5
```

### 交易成本
| 成本项 | 费率 | 方向 |
|--------|------|------|
| 滑点 | 0.2% | 双向 |
| 印花税 | 0.1% | 仅卖出 |
| 佣金 | 0.025% | 双向 |
